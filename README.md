# 75 Medium — My Challenge Tracker

A personal 75-day challenge tracker built as a single-file web app. Dark athletic theme, fully responsive, with localStorage so your progress saves automatically in your browser.

## Features
- **75-day grid** — tap to mark days complete, auto-highlights today
- **Daily checklist** — 6 rules, saves per-day automatically
- **Streak counter** — tracks consecutive completed days
- **Habit stack** — your full morning-to-night routine
- **Miss log** — honest daily reflection
- **Progress bar** — see the journey at a glance
- **Offline-ready** — no account, no server, just your browser

## Deploy to GitHub Pages (5 minutes)

### Step 1 — Create a new GitHub repo
1. Go to [github.com/new](https://github.com/new)
2. Name it something like `75-medium` or `my-challenge`
3. Set it to **Public**
4. Click **Create repository**

### Step 2 — Upload the file
1. On your new repo page, click **uploading an existing file**
2. Drag and drop `index.html` into the upload area
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to your repo **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **Deploy from a branch**
3. Set branch to **main**, folder to **/ (root)**
4. Click **Save**

### Step 4 — Access your site
Your site will be live at:
```
https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
```
(Takes ~60 seconds to go live after saving)

### Bookmark it!
Add it to your phone's home screen:
- **iPhone**: Open in Safari → Share button → "Add to Home Screen"
- **Android**: Open in Chrome → three dots menu → "Add to Home screen"

It'll look and feel like an app.

## Local use
Just open `index.html` in any browser — no server needed. Your data saves in that browser's localStorage.
